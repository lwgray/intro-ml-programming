Model Saving & Loading

Two ways to persist your trained Keras model

model.save() RECOMMENDED

Format: .keras

  • ✓ Architecture
  • ✓ Weights
  • ✓ Optimizer State
  • ✓ Training Config
# Save complete model
model.save("my_model.keras")

# Load — ready to use immediately
loaded = keras.models.load_model(
    "my_model.keras"
)
vs

model.save_weights()

Format: .weights.h5

  • ✗ Architecture
  • ✓ Weights
  • ✗ Optimizer State
  • ✗ Training Config
# Save weights only
model.save_weights(
    "my_weights.weights.h5"
)

# Load — must rebuild arch first!
new_model = build_model()
new_model.load_weights(
    "my_weights.weights.h5"
)
✅ Use model.save() for complete portability — one file, everything included
