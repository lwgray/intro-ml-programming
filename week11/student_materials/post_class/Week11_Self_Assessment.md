## Continued Learning Resources

These are the highest-leverage next steps:

### Courses
- **Hugging Face course (free):** https://huggingface.co/learn — comprehensive transformers
- **fast.ai courses:** http://fast.ai — practical deep learning
- **Stanford CS231n / CS224n:** http://cs231n.stanford.edu — deeper than this course

### Papers (start here)
- **Attention Is All You Need (2017):** the transformer paper
- **GPT-3 (2020):** scale laws and emergent abilities
- **InstructGPT (2022):** how RLHF works
- **Stable Diffusion / Latent Diffusion Models (2022)**

### Tools
- **LangChain / LlamaIndex:** LLM orchestration
- **Weights & Biases:** experiment tracking
- **Streamlit / Gradio:** quick ML app UIs
- **Modal / Replicate:** serverless ML deployment

### Communities
- **r/MachineLearning** (Reddit) — research discussions
- **Hugging Face Discord / Forums** — practical questions
- **Papers With Code:** https://paperswithcode.com — papers + implementations
