Filter Size & Weights: 3×3 vs 5×5
ELI5 · Convolutional networks

Why a 5×5 Filter Costs ~3× the Weights of a 3×3

With padding="same", both filters produce the exact same output size — that's the point of "same." What changes is the stamp itself: every output number is now a blend of 25 inputs instead of 9, and each of those 25 positions is a learned weight.

1 · Same output, bigger stamp

Padding keeps the map size; the kernel sets the weight count

Below, both convolutions compute the corner output cell of an 8×8 map. "Same" padding adds a ring of imaginary zeros (dashed) so the window can sit on the edge — a 3×3 filter needs a 1-cell ring, a 5×5 needs a 2-cell ring. Those zeros are not weights. The purple stamp is the weights.

Conv2D(k=(3,3), padding="same")

input 8×8 + zero ring
→
output 8×8
(same!)
9 weights + 1 bias
per filter, per input channel

Conv2D(k=(5,5), padding="same")

input 8×8 + two zero rings
→
output 8×8
(still the same!)
25 weights + 1 bias
per filter, per input channel — 2.8× the stamp area
The clarification worth saying out loud: padding never adds or removes weights — the zero ring is imaginary, not learned. Weights depend only on kernel size × channels. "Same" just guarantees both setups produce an identical 8×8 output, so the only difference left is the stamp area: 9 vs 25.

Notice the kernel area is a multiplier on channels-in × channels-out — so the 9-vs-25 gap gets amplified everywhere channels are wide. Students can verify every one of these numbers with model.summary().

3 · The modern trick

Two stacked 3×3s see 5×5 — for 18 weights, not 25

25 weights, one ReLU same view of the image, more parameters, less depth

This is why VGG (2014) and nearly everything since builds almost entirely from 3×3 filters: stacking small stamps buys the same view for fewer weights, plus an extra nonlinearity between them.

"Same" padding fixes the output size, so filter size becomes a pure price tag on the stamp: 3×3 pays 9 weights per channel for its view, 5×5 pays 25 for a view you could have built from two 3×3s at 18 — which is why small filters won.
