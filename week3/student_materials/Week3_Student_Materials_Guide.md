## ❓ FAQ

**Q: Do I need to complete pre-class materials?**  
**A:** STRONGLY recommended! Videos prime your brain. You'll get much more from live session.

**Q: What if I can't finish pair programming in 30 minutes?**  
**A:** That's OK! Focus on applying workflow, not perfection. Finish after class if needed.

**Q: Can I work with others on post-class assignment?**  
**A:** Discuss concepts together, but WRITE CODE INDEPENDENTLY. Copying = cheating yourself.

**Q: I'm stuck on post-class. What do I do?**  
**A:** 
1. Review Day 1 live session notebook (similar workflow)
2. Check Day 2 guided practice (simpler example)
3. Ask in forum (don't post full solutions!)
4. Office hours
5. THEN check solutions (learn from them)

**Q: How do I know if I'm ready for Week 4?**  
**A:** Complete self-assessment honestly. If you score <70%, review materials before Week 4.
